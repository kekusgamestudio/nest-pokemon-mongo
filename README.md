# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
$ yarn install
```
3. Tener nest cli instalado
```
$ npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
$ docker-compose up -d
```
5. Ejecutar el sistema en modo desarrollo
```
$ yarn start:dev
```
